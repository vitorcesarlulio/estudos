WEB 1
	Junto com os browsers de internet
	nessa versão eles funcionavam meio que um leitor de WebSite, do que um navegador de acesso a soluções
	Era uma via de mão única, onde os servers enviavam o conteúdo para os navegadores e os navegadores só mostravam esse conteúdo
	Não existia nenhuma interação
	
	Conteúdo estático
	Sites sem interação com usuário
	Netscape como principal navegador da época
	Isso por meados de 1991 quando Teams bernes li lançou o primeiro site do mundo
	
WEB 2
	Consumiam e interagiam
	não só likes e comentários, mas o próprio usuário inseria conteúdo que seria exibido
	
	Início dos anos 2000
	Nós termos dessa nova versão algo que se destaca é "confie no usuário"
	Onde os grandes players começaram crescer, Google, Amazon, eBay, Apple
	
WEB 3
	Desde de a WEB 1 havia uma centralização dos dados
	eu acesso o youtube para enviar e consumir vídeos
	é um ponto central de falha
	outro problema é privacidade, eu ceder algumas infos para acesso a algumas funcionalidades
	passar nome, e-mail, CPF, endereço
	As redes te controlam, para te dar acesso as funcionalidades
	
	Só existe por causa das blockchains
	ao invés de guardar nossos dados em um servidor de uma empresa a gente pode guardar nossos dados dentro de uma carteira ou de um smart contract dentro da blockchain 
	
	App para guardar anotações, essas infos que estão lá está no servidor da empresa que des. o app
	na web 3 você guardaria dentro de um smart contract na blockchain e ai você autorizaria o app a ter acesso a esse smart contract
	ele vai ser autorizado a acessar seu smart contract
	
	ele não armazena nada, só te fornece funcionalidades
