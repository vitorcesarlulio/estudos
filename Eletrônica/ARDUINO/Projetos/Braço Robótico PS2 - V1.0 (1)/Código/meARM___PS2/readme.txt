ESSE CODIGO POSSUI AS SEGUINTES FUNÇÕES:

BOTÃO START:   LIGA/DESLIGA - VIBRAÇÃO
BOTÃO SELECT:  SELECIONA MODO DE USO - ANALOGICO/BOTÕES

MOVIMENTAÇÃO POR BOTÕES:

BOTÃO R1/R2:   ABRE/FECHA  GARRA
BOTÃO O / []:  DIR/ESQ     BASE
BOTÃO /\ | X:  SOBE/DESCE  OMBRO
BOTÃO UP/DOWN: SOBE/DESCE  COTOVELO    (BOTÕES DO LADO ESQUERDO DO CONTROLE)

MOVIMENTAÇÃO POR JOYSTICK:

JOYSTICK ESQUERDO: CIMA/BAIXO = MOVIMENTA OMBRO
JOYSTICK ESQUERDO: DIR/ESQ    = MOVIMENTA BASE
JOYSTICK DIREITO:  CIMA/BAIXO = MOVIMENTA GARRA
JOYSTICK DIREITO:  DIR/ESQ    = MOVIMENTA COTOVELO
